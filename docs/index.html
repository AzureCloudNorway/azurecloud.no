About AzureCloud

Hi. My name is Sven Malvik and I am the organizers of Azure Meetup Oslo. We are one of several User Groups in Norway. Together, we want to provide you with interesting sessions and educational workshops, all about Azure and the tooling around.

AzureCloud is an initiative that was started by the organizers of the user groups to help you finding relevant Azure events. We put all events here having a common place; see it as an umbrella where our common denominator is Azure. We want to be as inclusive as possible, so please contribute and send us your events. We are happy to share them.

-Sven
